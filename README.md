# Car Parking Management System in C++

A simple menu-driven Car Parking Management System developed using C++ and File Handling.

## Features

* Park Vehicle
* Show All Vehicles
* Search Vehicle by Number
* Remove Vehicle
* Parking Fee Calculation
* Revenue Report
* Total Vehicles Count
* CSV File Storage
* Export Data to CSV

## Parking Charges

| Vehicle Type | Charge Per Hour |
| ------------ | --------------- |
| Bike         | ₹20             |
| Car          | ₹50             |
| SUV          | ₹80             |

## Technologies Used

* C++

* CSV Storage

## Data Storage

Vehicle records are stored in:

```text
vehicles.csv
```

Exported reports are stored in:

```text
parking_report.csv
```

These files can be opened directly in Microsoft Excel.
